Design System Facades

¶

Core Files

These files do not make up any controls themselves, but are the shared code across all (or most) Facades

Start with the Base core file.

¶

React Façade

These files are the controls implemented using the React library.

They are available as an AMD module at build/react.bundle.js in the root of this repository.

Start with the React index file to see what is provided.

The Lookup React control is particularly well documented.

¶

jQuery Façade

These files are controls for use by jQuery scripts.

They are available as an AMD module at build/jquery.bundle.js in the root of this repository.

They are also made available on the $.fn jQuery namespace as classic jQuery plugins.

Start with the jQuery index file to see what is provided.

¶

Miscellany

Some other interesting things in this project include a Data Adapter framework, Traits (for example the Keyboad Navigable Trait) and the Library that contains some cross-cutting utilities for the controls.
